**Style:** Dramatic / Conspiracy-core

## SCRIPT

---

**[HOOK — 0:00–0:05]**

**[OUTRO / CTA — 0:35–0:50]**

> *[End card: VoidPulse logo + subscribe prompt]*

## PRODUCTION NOTES

| Element | Details |
|---|---|
| **Voiceover pace** | Slow and deliberate — let numbers breathe |
| **Music** | Dark cinematic / Hans Zimmer-style tension build |
| **Visuals** | Red-tinted maps, falling dollar bills, blurred names |
| **Text on screen** | Bold white on black — slam each stat like a title card |
| **SFX** | Clock ticking, paper shredder, single heartbeat drop |
| **Hook retention trick** | Open on the number BEFORE saying what it is |
